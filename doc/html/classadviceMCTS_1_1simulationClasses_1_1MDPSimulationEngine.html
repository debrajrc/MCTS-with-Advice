Detailed Description

MDPSimulationEngine is used to draw simulations that extend an execution, according to an action strategy.

The execution is sequentially extended and reset after each simulation, Use execution.executionCopy() if running several MDPSimulationEngine instances in parallel to give independent instances to the constructors.
