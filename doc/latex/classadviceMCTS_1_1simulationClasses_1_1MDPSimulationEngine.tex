\doxysubsection{Detailed Description}
\mbox{\hyperlink{classadviceMCTS_1_1simulationClasses_1_1MDPSimulationEngine}{M\+D\+P\+Simulation\+Engine}} is used to draw simulations that extend an execution, according to an action strategy. 

The execution is sequentially extended and reset after each simulation, Use execution.\+execution\+Copy() if running several \mbox{\hyperlink{classadviceMCTS_1_1simulationClasses_1_1MDPSimulationEngine}{M\+D\+P\+Simulation\+Engine}} instances in parallel to give independent instances to the constructors.
